# 📝 ToDo List App

A simple and clean ToDo List application built with React Native.  
Manage your daily tasks easily with a minimalistic interface. 🎯

## ✨ Features

- ✅ Add new tasks  
- ✔️ Mark tasks as completed  
- 📋 View task list  
- 🎨 Clean and responsive UI

React Native

JavaScript

Expo CLI
